PUBLISH CONTRACT

START CHECKS

- Never discard dirty work.

BRANCH INVENTORY
- List all local deck/* branches.
- Classify each as already merged, safe candidate, conflict, risky/delete/trash, likely inactive-variant edit, unrelated, broken ref, or unknown.

- Prefer branches whose final report or commit evidence indicates active-slide verification.

- Skip delete/trash/risky branches.
- Skip conflicts.
- Merge only verified safe completed deck/* branches Dave asked to publish or clearly approved.

- Commit publish changes on main.
- Push only when Dave explicitly asked to publish.
