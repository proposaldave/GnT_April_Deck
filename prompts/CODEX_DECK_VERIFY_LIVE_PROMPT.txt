CLASSIFY THE ISSUE
- cache: raw GitHub and cache-busted Pages have the change, canonical browser view is stale.
- source not published: local or raw source lacks the change.
- inactive variant: change exists only in retained/deleted/appendix/stale HTML, not active registry path.
- skipped branch: change exists only on an unmerged branch.
- conflict: branch was not merged because it conflicts.
- wrong slide: edit landed on a different active slide/slot than Dave requested.
- rendered hidden: source exists but visible render hides, covers, or does not reach it.
- request ambiguity: requested target cannot be mapped to one active slide/section.
- browser state: browser tab/session is stale even though cache-busted source is current.
- visual mismatch: source contains the change but rendered layout does not match intent.
